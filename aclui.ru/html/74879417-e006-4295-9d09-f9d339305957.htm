Страница свойств параметров безопасности

Для обеспечения безопасности компьютера и его ресурсов необходимо учитывать права, которыми будут располагать пользователи. Безопасность одного или нескольких компьютеров можно обеспечить, предоставив пользователям или группам особые права пользователя. Безопасность объекта, например, файла или папки, можно обеспечить, назначив пользователям и группам разрешения для выполнения особых действий над данным объектом.

Параметр Описание

Имя объекта

Называет текущий выбранный объект.

Группы или пользователи

Отображает список групп или пользователей, которым предоставлен доступ к данном компьютеру.

Изменить

Разрешает изменение параметров безопасности для данного объекта.

Примечание

Если эта кнопка недоступна, вероятно, отсутствуют разрешения на изменение параметров безопасности. Чтобы их просмотреть, можно нажать кнопку Дополнительно.

Разрешения для

Отображает список действующих разрешений и запретов разрешений для каждой выбранной группы или пользователя.

Примечание

Указанные типы разрешений будут зависеть от типа выбранного объекта. Дополнительные сведения см. в статье Управление разрешениями(http://go.microsoft.com/fwlink/?LinkId=141450).
