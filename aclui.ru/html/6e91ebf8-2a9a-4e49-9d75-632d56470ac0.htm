Определение применения разрешений

Если для файлов и папок установлены дополнительные разрешения, в диалоговом окне Запись разрешений для <имя объекта> можно выбрать дочерние объекты, которые будут наследовать разрешения.

Чтобы найти это диалоговое окно, нажмите кнопку Дополнительно в пользовательском интерфейсе управления доступом. На вкладке Разрешения дважды щелкните Изменить.

В диалоговом окне Запись разрешений для <имя объекта> , в поле Применять на вкладке Объект перечислены расположения, где можно применить разрешения. Способ применения разрешений зависит от того, установлен ли флажок Применять эти разрешения к объектам и контейнерам только внутри этого контейнера.

Важно!

При работе с объектами Active Directory записи управления доступом (ACE) наследуют не только объекты, указанные в поле Применять, но все дочерние объекты также получают копии данных записей. Дочерние объекты, не указанные в поле Применять, не используют записи управления доступом, копии которых они получают, но при наличии достаточно большого числа объектов, получающих копии данных записей, количество данных возрастает, что может вызвать значительное снижение производительности сети.

По умолчанию этот флажок снят. В следующей таблице указаны детальные сведения о применении наследуемых разрешений:

  • Если флажок «Применять эти разрешения к объектам и контейнерам только внутри этого контейнера» снят

  • Если флажок «Применять эти разрешения к объектам и контейнерам только внутри этого контейнера» установлен
