View the Security Log

The Security log records each event as defined by the audit policies you set on each object.

To view the Security log

  1. Open Event Viewer.

  2. In the console tree, open Global Logs, and then click Security. The results pane lists individual security events.

  3. If you want to see more details about a specific event, in the results pane, double-click the event.

Additional considerations

  • To open Event Viewer, click Start, click Control Panel, click System and Maintenance, double-click Administrative Tools, and then double-click Event Viewer.

  • If your computer is connected to a network, network policy settings might also prevent you from completing this procedure.
